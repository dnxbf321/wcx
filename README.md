# wcx - stand for wei cheng xu
微信小程序脚手架，支持 npm、ES6、postcss、eslint

# Usage

### 安装

```bash
npm install wcx -g
wcx --help
```

### 使用

请参考 [wcx-template](https://github.com/dnxbf321/wcx-template#readme) package.json scripts
